# Compound Engineering Settings

This file configures the review agents for the `/ce:review` command.

## Default Review Agents

- **security-sentinel**: Security audits, vulnerability detection, OWASP compliance
- **performance-oracle**: Performance analysis, bundle size, runtime efficiency
- **architecture-strategist**: Architectural patterns, component design, code organization
- **pattern-recognition-specialist**: Design patterns, naming conventions, code consistency
- **kieran-typescript-reviewer**: TypeScript best practices, type safety, maintainability
